# CodePen Clone

This project is a CodePen Clone. I made this project to improve myself at React. This project was made with React. Original version of the [Web Dev Simplified](https://www.youtube.com/watch?v=wcVxX7lu2d4&list=LL&index=11&t=16s&ab_channel=WebDevSimplified)

## Live Demo

You can look at project demo in [here](https://yigiterdrive.vercel.app/login).

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```
